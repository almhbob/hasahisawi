## الفكرة والدافع

حصاحيصا مدينة سودانية بتاريخ عريق في ولاية الجزيرة، يمتد أبناؤها عبر أرجاء السودان والمهجر. لكن لم يكن ثمة منصة رقمية موحدة تجمعهم — فكانت المعلومات متفرقة، والتواصل صعباً، والخدمات غير موثقة.

**حصاحيصاوي** وُلد ليكون الجسر الرقمي بين أبناء المدينة؛ منصة واحدة تجمع الأخبار، والخدمات، والتواصل، والتوثيق المجتمعي — يستخدمها المقيم والمغترب على حدٍّ سواء.

## مراحل التطوير

### المرحلة الأولى — التأسيس والهيكل
- إنشاء بنية المشروع (Expo + React Native + TypeScript)
- تصميم قاعدة البيانات (PostgreSQL)
- بناء خادم API (Express + Node.js)
- نظام تسجيل الدخول والمصادقة
- شاشات التسجيل وإنشاء الحساب
- تحديد هوية التطبيق البصرية (ألوان، خطوط، أيقونة)

### المرحلة الثانية — الميزات الأساسية
- شاشة المنزل مع الخدمات السريعة
- دليل المستشفيات والعيادات
- دليل المدارس والمؤسسات التعليمية
- دليل الأندية الرياضية
- دليل المراكز الثقافية
- بطاقة الأرقام المهمة
- قسم المفقودات والضائع

### المرحلة الثالثة — المحتوى الاجتماعي
- الخلاصة الاجتماعية (منشورات + إعجابات + تعليقات)
- نظام الدردشة المباشرة (محادثات ثنائية)
- الإشعارات الفورية
- منصة الإعلانات المدفوعة
- دليل المؤسسات مع خريطة وتقييمات
- خدمات المرأة

### المرحلة الرابعة — التفاعل والبيانات
- نظام المجتمعات والمجموعات
- بلاغات المواطنين
- نظام المواعيد
- المساعد الذكي (AI Support)
- سوق المنتجات
- لوحة تحكم المشرفين

### المرحلة الخامسة — التجربة والاحترافية
- شاشة الإعداد الأولي (Onboarding) بـ 4 شرائح
- استعادة كلمة المرور (3 خطوات)
- الملف الشخصي مع تعديل السيرة وإحصائيات
- البحث الشامل (مستخدمون + منشورات + مؤسسات + مجتمعات)
- رفع وتحديث أيقونة التطبيق
- تحسين التخطيط والمسافات

## الميزات الكاملة للتطبيق

### 🏠 الشاشة الرئيسية
- إعلانات دوّارة (Banner Ads)
- أزرار الخدمات السريعة بتصميم grid
- قسم "الخدمات" مع 12+ فئة
- مختصرات سريعة لأبرز الأقسام
- تحديث تلقائي للمحتوى

### 📰 الخلاصة الاجتماعية (Social Feed)
- نشر منشورات نصية وصور وفيديو
- إعجاب وتعليق على المنشورات
- تصفية حسب الفئة (أخبار، رياضة، ثقافة، إلخ)
- عرض المنشورات بالترتيب الزمني
- منشورات مُميزة للمؤسسات
- دعم الإعلانات المدفوعة بين المنشورات

### 💬 نظام الدردشة
- محادثات ثنائية مباشرة
- قائمة المحادثات مع آخر رسالة
- مؤشر الرسائل غير المقروءة
- واجهة على طراز Messenger
- دعم الرسائل النصية

### 🔍 البحث الشامل
- البحث في: المستخدمين، المنشورات، المؤسسات، المجتمعات
- نتائج من قاعدة البيانات المحلية والسيرفر
- عرض الصورة الشخصية في نتائج المستخدمين
- انتقال مباشر للملف الشخصي عند اختيار مستخدم
- بحث في: المستشفيات، المدارس، الأندية، المراكز، الأرقام، المفقودات

### 👤 الملف الشخصي
- عرض البيانات الشخصية (الاسم، الدور، تاريخ الانضمام)
- تعديل السيرة الذاتية
- إحصائيات: عدد المنشورات، البلاغات
- عرض آخر المنشورات
- إعدادات الحساب
- زر "مراسلة" عند زيارة ملف شخص آخر

### 🏥 دليل الخدمات الطبية
- قائمة المستشفيات والعيادات
- أرقام التواصل
- معلومات التخصصات
- خيارات الاتصال المباشر أو واتس آب

### 🎓 الدليل التعليمي
- المدارس الحكومية والخاصة
- معلومات الاتصال والعنوان
- المراحل الدراسية

### ⚽ الدليل الرياضي
- الأندية الرياضية
- الملاعب والمرافق
- معلومات التواصل

### 🎭 المراكز الثقافية
- المكتبات والمراكز
- الفعاليات والأنشطة

### 🏛️ دليل المؤسسات
- مؤسسات حكومية وأهلية
- خريطة وموقع جغرافي
- نظام التقييم (1-5 نجوم)
- طلب الانضمام للمؤسسات
- فلترة حسب الفئة والموقع

### 👥 المجتمعات
- إنشاء وإدارة مجتمعات
- الانضمام بالدعوة أو عامة
- منشورات خاصة بالمجتمع

### 📢 بلاغات المواطنين
- رفع بلاغ مع الموقع والوصف
- تتبع حالة البلاغ
- أنواع البلاغات: أمن، بنية تحتية، صحة، بيئة

### 📅 المواعيد
- حجز مواعيد مع الخدمات
- تتبع المواعيد القادمة
- تذكيرات

### 🤖 المساعد الذكي
- إجابات على الأسئلة المتعلقة بالمدينة
- دعم بالعربية الكاملة

### 🛒 السوق
- إعلانات البيع والشراء
- فئات المنتجات المتعددة

### 📊 الإعلانات المدفوعة
- إعلانات البانر في الشاشة الرئيسية
- إعلانات مدمجة في الخلاصة
- إدارة الإعلانات من لوحة التحكم

### ⚙️ لوحة تحكم المشرف
- إدارة المستخدمين
- مراجعة البلاغات
- إدارة الإعلانات
- الإعدادات العامة
- إشعارات جماعية

### 🔔 الإشعارات
- إشعارات فورية للأحداث المختلفة
- قائمة الإشعارات مع تاريخ وأيقونة
- تمييز المقروء/غير المقروء

### 👩 خدمات المرأة
- قسم مخصص لخدمات وأنشطة المرأة
- معلومات الصحة والتعليم والتوعية

### 📞 الأرقام المهمة
- أرقام الطوارئ
- دوائر الخدمات الحكومية
- اتصال مباشر بنقرة واحدة

## واجهة API — أبرز نقاط النهاية

```
POST   /api/auth/register          تسجيل مستخدم جديد
POST   /api/auth/login             تسجيل الدخول
POST   /api/auth/logout            تسجيل الخروج
POST   /api/auth/check-phone       فحص رقم الهاتف
POST   /api/auth/forgot-password   استعادة كلمة المرور

GET    /api/posts                  الخلاصة الاجتماعية
POST   /api/posts                  نشر منشور جديد
POST   /api/posts/:id/like         إعجاب بمنشور
POST   /api/posts/:id/comments     تعليق على منشور

GET    /api/chats                  قائمة المحادثات
POST   /api/chats                  بدء محادثة جديدة
GET    /api/chats/:id/messages     رسائل محادثة
POST   /api/chats/:id/messages     إرسال رسالة

GET    /api/organizations          قائمة المؤسسات
GET    /api/communities            قائمة المجتمعات
GET    /api/citizen-reports        قائمة البلاغات
POST   /api/citizen-reports        رفع بلاغ جديد

GET    /api/search?q=...           البحث الشامل
GET    /api/users/:id/profile      ملف مستخدم عام
PATCH  /api/users/me/bio           تحديث السيرة الذاتية

GET    /api/ads                    الإعلانات النشطة
GET    /api/notifications          الإشعارات
GET    /api/landmarks              المعالم الجغرافية
```
